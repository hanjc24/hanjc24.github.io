单层Model

Model: y^=fθ(x)\hat{y}=f_{\theta}(x)y^​=fθ​(x)
Loss: L=L(y,y^)L=L(y,\hat{y})L=L(y,y^​)
Optimize: minimize L=L(y,fθ(x))L=L(y,f_\theta(x))L=L(y,fθ​(x)) by changing θ\thetaθ
方法是梯度下降：∂L∂θ=∂L∂y^∂f∂θ\frac{\partial L}{\partial\theta}=\frac{\partial L}{\partial\hat{y}}\frac{\partial f}{\partial\theta}∂θ∂L​=∂y^​∂L​∂θ∂f​

多层Model

Model: y^=fθnnfθn−1n−1...fθ11(x)\hat{y}=f^{n}_{\theta_n} f^{n-1}_{\theta_{n-1}} ... f^{1}_{\theta_1} (x)y^​=fθn​n​fθn−1​n−1​...fθ1​1​(x) (嵌套)
note: 记fif_ifi​的输入为xix_ixi​，输出为xi+1x_{i+1}xi+1​，则x=x1x=x_1x=x1​，y^=xn+1\hat{y}=x_{n+1}y^​=xn+1​

Data: input xxx and expected output yyy (label)
Loss: L=L(y,y^)L=L(y,\hat{y})L=L(y,y^​)
Optimize: minimize L=L(y,fθnfθn−1...fθ1(x))L=L(y,f_{\theta_n}f_{\theta_{n-1}}...f_{\theta_1}(x))L=L(y,fθn​​fθn−1​​...fθ1​​(x)) by changing θ\thetaθ using gradient descent
因此需要高效计算：∂L∂θi∣x=xi,θ=θi\frac{\partial L}{\partial\theta_i}|_{x=x_i,\theta=\theta_i}∂θi​∂L​∣x=xi​,θ=θi​​

Forward

已知条件：θi\theta_iθi​和Data
直接计算得到xix_ixi​与LLL
因为计算的顺序是 iii 从 1 到 n，所以是前向传播

Backpropagation

已知条件：∂fi∂θi∣x=xi,θ=θi,∂fi∂xi∣x=xi,θ=θi,∂L∂y^\frac{\partial f^i}{\partial\theta_i}|_{x=x_i,\theta=\theta_i},\frac{\partial f^i}{\partial x_i}|_{x=x_i,\theta=\theta_i},\frac{\partial L}{\partial\hat{y}}∂θi​∂fi​∣x=xi​,θ=θi​​,∂xi​∂fi​∣x=xi​,θ=θi​​,∂y^​∂L​
前两者由fif^ifi层与fif_ifi​层当前的参数和前传输入决定
后者由Loss函数、Label和当前前传结果y^\hat{y}y^​决定
上述这些条件的具体数值会在前传的过程中被确定下来

用链式求导法则归纳得到∂L∂θi∣x=xi,θ=θi\frac{\partial L}{\partial\theta_i}|_{x=x_i,\theta=\theta_i}∂θi​∂L​∣x=xi​,θ=θi​​

  • Basis:
    ∂L∂θn=∂L∂y^∂fn∂θn\frac{\partial L}{\partial\theta_n}=\frac{\partial L}{\partial\hat{y}}\frac{\partial f^n}{\partial\theta_n}∂θn​∂L​=∂y^​∂L​∂θn​∂fn​

    ∂L∂xn=∂L∂y^∂fn∂xn\frac{\partial L}{\partial x_n}=\frac{\partial L}{\partial\hat{y}}\frac{\partial f^n}{\partial x_n}∂xn​∂L​=∂y^​∂L​∂xn​∂fn​
  • Induction: 方向为iii从nnn到111（反向传播）
    对于第iii层fif_ifi​，已知∂L∂xi+1\frac{\partial L}{\partial x_{i+1}}∂xi+1​∂L​，xi+1=fθii(xi)x_{i+1}=f^i_{\theta_i}(x_i)xi+1​=fθi​i​(xi​)
    ∂L∂θi=∂L∂xi+1∂fi∂θi\frac{\partial L}{\partial\theta_i}=\frac{\partial L}{\partial x_{i+1}}\frac{\partial f^i}{\partial\theta_i}∂θi​∂L​=∂xi+1​∂L​∂θi​∂fi​

    ∂L∂xi=∂L∂xi+1∂fi∂xi\frac{\partial L}{\partial x_i}=\frac{\partial L}{\partial x_{i+1}}\frac{\partial f^i}{\partial x_i}∂xi​∂L​=∂xi+1​∂L​∂xi​∂fi​

    再次明确，在反向传播的过程中，进行的是单纯的计算，算的是在前传确定的位置上，LLL关于θi\theta_iθi​的偏导的具体数值，计算机并不需要理解各类求导规则来计算公式，只需要应用链式求导法即可

一点补充：pytorch的计算图抽象

对于一个神经网络，可以把它抽象为一个DAG(有向无环图)，称为计算图。叶子节点是输入量，根是输出量，叶子以外的节点都是函数。边用来前传和梯度反传
训练时，这个DAG的唯一一个没有出度的节点就是最终的loss，一般是一个标量，记为l=g(y⃗)l=g(\vec{y})l=g(y​)。
这是一个示意的计算图图，这里A、B是输入值(参数或数据)，l是输出值，fff、ggg、hhh则是函数

graph LR
A -- x1 --> f
B -- x2 --> f
f -- y1 --> g
f -- y2 --> h
g -- z1 --> l
h -- z2 --> l

这张图对应代码： y1, y2 = f(x1, x2)、z1 = g(y1)、z1 = g(y1)、z2 = g(y2)、l = l(z1, z2)
y⃗=f⃗(x⃗)\vec{y}=\vec{f}(\vec{x})y​=f​(x) 这里函数fff是确定的，不含未知参数。xxx为fff节点所有入边(指向fff的节点的输出向量)拼接成的向量，yyy被拆分后为fff的所有出边的前传输出向量
前传过程中将x⃗∈Rn\vec{x}\in \mathbb{R}^nx∈Rn确定后，y⃗∈Rm\vec{y}\in \mathbb{R}^my​∈Rm以及Jf∈Rm×nJ_f\in \mathbb{R}^{m\times n}Jf​∈Rm×n在x⃗\vec{x}x处的具体值也随之确定。

反传时，我们有 ∂l∂y∈Rm\frac{\partial l}{\partial y} \in \mathbb{R}^m∂y∂l​∈Rm 作为传递到a节点值的梯度(从a节点的出边传回来的梯度之和)
而且根据求导法则，∂l∂xi=∑j=1m∂l∂yj∂yj∂xi\frac{\partial l}{\partial x_i}=\sum_{j=1}^m\frac{\partial l}{\partial y_j}\frac{\partial y_j}{\partial x_i}∂xi​∂l​=∑j=1m​∂yj​∂l​∂xi​∂yj​​，写成矩阵的形式就得到 ∂l∂x=JfT∂l∂y\frac{\partial l}{\partial x}=J_f^T\frac{\partial l}{\partial y}∂x∂l​=JfT​∂y∂l​
这样得到∂l∂x\frac{\partial l}{\partial x}∂x∂l​后，再将这个梯度切分并传给所有指向节点a的节点
反传的过程类似拓扑排序：

  • 从最终的loss开始(loss关于自己的梯度为1)，每次处理一个没有出度的节点(其各个输出向量的梯度已经都计算完毕，即得到了完整的∂l∂y\frac{\partial l}{\partial y}∂y∂l​)
  • 用上面的求导法则计算出∂l∂x\frac{\partial l}{\partial x}∂x∂l​，得到每个输入量的梯度
  • 反传这些梯度到指向该节点的所有节点的输出张量的梯度上
  • 把这个节点和指向这个节点的边删除(指向该节点的所有节点出度-1)
  • 如果还有节点未被处理，回到第一步，否则结束算法，所有输入量的梯度均已计算完毕

在pytorch中，这样的fff被实现为torch.autograd.Function对象，该对象要实现两个函数(或者说接口/方法)：

  • forward(x1, x2, ..., xn) -> Tuple(y1, ... ,ym)
  • backward(grad_y1, ... ,grad_ym) -> Tuple(grad_x1, ..., grad_xn)
    这里的形式参数和返回值都是torch.tensor。pytorch会根据计算过程自动构建计算图，调用各个节点的forward和backward进行前传和反传
    备注：实际使用中，一个输出张量可能还会作为后面多个torch.autograd.Function的输入张量被使用，反传时统计该输出张量的梯度，只要将反传回来的多个梯度累加即可。

参考链接：pytorch入门指引
