大一暑假(2025.7)时学习了台湾大学李宏毅老师的机器学习课程，今天重温了一下课程内容，顺便把当时记的笔记更新上来。

Regression

How Machine learns?

how to find this function ?： Training based on training data.(ML is data driven)

  1. Assume a Function with Unknown Parameters (Model, based on domain knowledge)
    例如
    feature(特征)
    & weight and bias(unknown, will be learned from data)
    这个例子是线性回归

  2. Define Loss from Training Data
    Loss is a function of parameters in the Model
    Loss: how good a set of values is (损失函数)

  3. Optimization: find 损失函数最小化
    Gradient Descent

    • (Randomly) Pick an initial value
    • 计算关于的偏导
    • 调大(偏导小于0)或调小(偏导大于0)
    • Learning rate。hyperparameters 人为提前设定的参数
    • 迭代，局部最优解。
      多个参数同理

  4. Generalization and Application: 用训练出的模型与参数预测新的数据等

    • 回归 The function outputs a scalar
    • 聚类或者说选择 Given classes, the function outputs the correct one
    • create something with structure(image, document)

How to choose a Model?(stage 1)

观察规律，基于已有的知识。譬如用七点前的数据预测，譬如用前七天的数据加权预测，譬如不再使用线性函数作为模型
Linear models have severe limitation: Model Bias
need more flexible model

如何更高效地计算Loss：
将所有已有数据N分成多个batch，对于一个可以用第i个batch决定的计算出的更新(update)当前的
batch size * batch num = N
依次用第1个batch到最后一个batch更新的过程=1 epoch

More Complex Model：神经网络



这样迭代的次数被称为神经网络的层数(layer)
而每个激活函数都被称为神经元Neuron，构成神经网络Neural Network
在神经网络的感知层和输出层中间的层数是hidden layer
Many layers means Deep Deep Learning

Layer/Deep的用处——为什么不用一层，用很多的ReLu/sigmoid逼近？搞一个Fat network
单层过多的sigmoid应该也会导致过拟合吧？
